Coopr Documentation

Coopr is a software system that enables end users to provision clusters based on templates and settings defined by administrators. Coopr exposes simple interfaces to administrators and allows easy customization of the properties of provisioned clusters, including software services, hardware configurations and infrastructure providers.

Coopr User Groups and Mailing Lists

  • coopr-user@googlegroups.com

The coopr-user mailing list is primarily for users using the product to deploy clusters. You can expect questions from users, release announcements, and any other discussions that will be helpful to the users.

  • coopr-dev@googlegroups.com

The coopr-dev mailing list is essentially for developers actively working on the product, and should be used for all design, architecture, and technical discussions. This mailing list will also receive all JIRA and GitHub notifications.
